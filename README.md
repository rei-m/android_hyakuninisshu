百人一首
========

百人一首の暗記用アプリです。4択から正解を選ぶ形式で繰り返し練習できます。

## Contents
- 練習
- 力試し
- 資料
- 他、希望あればIssueに書いておくと対応するかもしれません

## Environment
- 開発言語 : Java -> Kotlin
- min sdk : 18

## Structure
- MVVM + Fluxな構成になっています。が、仕様に対して過剰な設計です。個人の設計の遊び場としても作っているのであまり参考にはしない方がいいと思います。
